Common Features
This is GD Workspace, but Demo focused on Dashboard, Windows and Graphics GD components.
 • Dashboard content is GDWindow collection.
 • Dashboard has ability to reorder children (just drag and drop).
 • Dashboard auto layout content.
 • Dashboard has bottom thumbnail panel, you expand collepse it (in demo the panel is collepsed).
    Each GDWindow on dashboard has minimize button.
    Minimize window means move, it to thumbnail panel.
    Window can be restored any time, by thumbnail click.
 • GDGraph uses Chart.js library behind.
